# Changelog

All notable changes to **Project Cassandra** will be documented in this file.

## [v4.0.0] - 2026-01-07
### Added
- **Sanitized Documentation:** Rebranded as an "OSINT Decision Support Framework" for compliance and professional presentation.
- **Project Structure:** Standardized folder structure (`src/`, `docs/`, `examples/`).
- **Safety Rails:** Added operational security protocols to `system_prompt.md`.
- **License:** Added MIT License.

### Changed
- **System Prompt Refactoring:** Implemented "Internal Code" mappings (e.g., mapping "Alpha-Seeker" to "Degen Logic") to maintain LLM performance while sanitizing visible terminology.
- **Liquidity Logic:** Refined "Zombie Market" and "Imbalance" filters for greater precision.

## [v3.6.0] - 2026-01-07
### Added
- **Portfolio Hedge Logic:** Mechanism to calculate hedge ratios for global risks.
- **Market Making Mode:** Protocol for liquidity provision in low-volume markets.

## [v3.4.0] - 2026-01-07
### Added
- **Dead-End Detector:** Kill switch for contract expiration risks (e.g., Runoffs).
